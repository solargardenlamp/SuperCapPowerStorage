Welcome to GitHub Pages!

This is a fully self-contained website, generated from a single file (`index.html`). It's ready to be hosted on your new GitHub Page.

The content is styled with Tailwind CSS for a professional, responsive look.

Next Steps

  1. Create a new public repository on GitHub.
  2. Add this `index.html` file to the root of that repository.
  3. Go to **Settings** > **Pages** and set the source branch to `main` (or `master`).
  4. Wait a minute, and your site will be live!
